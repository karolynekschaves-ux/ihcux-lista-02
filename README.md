# Prático 2: O "Terminal de Suporte Proativo" 🛠️
Cenário:
Você foi contratado por uma empresa de TI para criar o Console de Diagnóstico de
Rede. O sistema atual é terrível: o técnico precisa digitar comandos enormes sem
errar uma letra. Se ele errar, o sistema apenas diz "Command Error" e apaga tudo.
## O Desafio:🎯
Desenvolver um protótipo em C# que ajude o técnico a diagnosticar um servidor,
focando em Acessibilidade e Prevenção de Erros.
